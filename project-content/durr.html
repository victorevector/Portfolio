Durr-inspired Watch

I built a watch that vibrates every 5 minutes.
It is a thought experiment on time's simultaneous objectivity and subjectivity.

For this project I learned to...
+ program and prototype circuits with the Arduino
+ program the ATTiny 85 micro controller
+ design printed circuit boards (PCBs) with Eagle software
+ solder components onto a PCB
+ make creative use of colorful electrical tape
