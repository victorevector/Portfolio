Genetics and Drug Metabolism

In many cases, a persons genetic code can offer insight into how that person will metabolize and react to certain medications. I built an application that can inform a user whether he or she carries genetic markers correlated to certain drugs. For example, a user could find out if they carry a sensitivity to the drug, Carbemazepine. The app will inform the user if their DNA contains the single-nucleotide polymorphism (SNP), rs1061235(T). Carriers of that SNP have an increased risk of Carbamazepine-induced hypersensitivity reactions, such as maculopapular exanthema.

I built the app with python, django, and postgres. The user's data is hosted on the 23andMe servers, which requires an Oauth2 handshake to access. I built out the Oauth2 client from scratch.
